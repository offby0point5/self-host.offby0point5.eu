The story so far

2024-04-06
:: tags:  #homelab  #introduction

I started my self-hosting journey with a Minecraft server somwhere between 2020 or 2021 on my laptop. This was less than ideal, given that I had to run the server and my client on the same T450 ThinkPad.

A bit later I've got a PC so the server could reside there and I only would have to send the IP address through Discord if it had changed so everyone would know how to connect to the server again. This problem was solved, when I discovered the free nsupdate.info service.

At some point, the CMOS battery died and the PC refused to boot up afer a disconnect from the mains. I took the opportunity to replace the Debian system with Proxmox VE and migrate the Minecraft server into an LXC container.

As I dropped out of university, I've got plenty of time to discover all the things one could self-host too.

When starting a new course of studies the year after, I also became part of the students union on my university. They host their website and tools on their own hardware in the basement of the university. This is where I learned about NixOS and started to replace all of the Alpine/Debian containers with ones using NixOS.

Later on I integrated all of the container configurations into one Nix flake.

I will probably write about my homelab in more than one post in the future, so stay tuned!
